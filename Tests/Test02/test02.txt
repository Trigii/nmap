Authors: Javier Sande Ríos; Tristán Vaquero Povedano

Description: Scan host with a valid and not blacklisted self-signed certificate with weak algorithms (MD5) and short validity (less that one month).

Input:
	* list: blacklist.csv
	* dns-list: dns.csv

Output:
	* Information about the certificate
	* Self-signed: True
	* Verified: True
	* Validity: True
	* Valid names: True
	* Blacklisted: False
	* DNS resolved: True
	* DNS: IP in valid range
	* Warnings:
		* Weak algorithms
		* Short validity

Command: $ nmap -Pn -p443 10.0.3.149 --script untrustedX509certs.nse --script-args list=blacklist.csv dns-list=dns.csv
