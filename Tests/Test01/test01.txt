Description: Scan host with a valid and not blacklisted self-signed certificate with no security warnings (Certs/Test01/server.pem) and ip in a valid range.

Input:
	* list: blacklist.csv
	* dns-list: dns.csv

Output:
	* Information about the certificate
	* Self-signed: True
	* Verified: True
	* Validity: True
	* Valid names: True
	* Blacklisted: False
	* DNS resolved: True
	* DNS: IP in valid range


Command: $ nmap -Pn -p443 10.0.3.149 --script untrustedX509certs.nse --script-args list=blacklist.csv dns-list=dns.csv -v -oN test02.txt
