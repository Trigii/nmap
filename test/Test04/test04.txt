Authors: Javier Sande Ríos; Tristán Vaquero Povedano

Description: Scan host with a valid certificate signed by a valid CA. None of them blacklisted.
 
Input:
	* list: blacklist.csv
	* dns-list: dns.csv

Output:
	* Information about the certificate
	* Self-signed: False
	* Verified: True
	* Validity: True
	* Valid names: True
	* Blacklisted: False
	* DNS resolved: True
	* DNS: IP in valid range

Command: $ nmap -Pn -p443 10.0.3.149 --script untrustedX509certs.nse --script-args list=blacklist.csv dns-list=dns.csv
