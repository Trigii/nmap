Authors: Javier Sande Ríos; Tristán Vaquero Povedano

Description: Scan host with a valid and with blacklisted certificate fingerprint and not valid range.

Input:
	* list: blacklist.csv
	* dns-list: dns.csv

Output:
	* Information about the certificate
	* Self-signed: True
	* Verified: True
	* Validity: True
	* Valid names: False
	* Blacklisted: True (Blacklisted subject)
	* DNS resolved: True
	* DNS: IP NOT in valid range

Command: $ nmap -Pn -p443 10.0.3.149 --script untrustedX509certs.nse --script-args list=blacklist.csv dns-list=dns.csv -v -oN test10.out
